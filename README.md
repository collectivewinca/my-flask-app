# My Flask App

Flask application workspace for backend feature and API experimentation.

## Purpose

This repository is used to build and test Flask-based functionality, routes, and integrations.

## Expected Experience

- Quick local startup for backend development
- Iteration on endpoints and app logic
- Lightweight environment for prototype validation

## Getting Started

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Notes

Adjust startup command based on the project entry file if different from `app.py`.
